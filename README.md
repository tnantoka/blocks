# The Blocks of the Dead

Typing Objective-C Blocks.

## Installation

```
$ git clone git@github.com:tnantoka/blocks.git
$ cd blocks
$ npm install
```

## Run

```
$ npm start
```

## Test

```
# unit
$ npm test # unit

# e2e
$ npm run update-webdriver
$ npm run protractor
```
